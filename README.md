🧩 Defense Tower Game (Unity)

A simple Tower Defense game developed using Unity and C#. This project was built as part of my learning process in game development, focusing on understanding core gameplay mechanics and Unity fundamentals.

🎮 Features
Enemy spawning system with wave-based logic
Tower placement and attack mechanics
Basic AI movement for enemies
Collision detection using Unity Collider
Simple UI elements for gameplay interaction

🛠️ Technologies Used
Unity Engine
C#
MonoBehaviour scripting
Prefabs, Colliders, Rigidbody

🚀 What I Learned
Understanding Unity project structure and workflow
Implementing gameplay logic using C# scripts
Working with core Unity components (Update, Prefabs, Collisions)
Debugging and modifying existing game systems
Adjusting gameplay parameters such as speed, spawn rate, and interactions

📌 Notes
This project was initially based on a tutorial to learn Unity fundamentals. I further explored and modified parts of the project to better understand how core game systems work.

▶️ How to Run
Clone the repository
Open with Unity Hub
Select the project folder
Run the game in Unity Editor
